Purpose
-------
| Microsoft introduced `Microsoft Entra <https://www.microsoft.com/en-us/security/business/microsoft-entra>`__ as a new product family for all the Azure identity and access products, bringing to life the Azure expanded vision for secure access.
| At that time, the Microsoft Entra product family included Azure AD and two new products. Microsoft Entra Permissions Management, a cloud infrastructure entitlement management tool, and Microsoft Entra Verified ID, a decentralized identity solution.

| Microsoft Entra unifies identity and access capabilities, simplifying navigation and use of all products in the portfolio. It ensures the strongest protections for any access point.
| The standardization of the use of the term “ID” is for all the Azure identity products:
| Microsoft Entra ID, ID Protection, ID Governance, External ID, Workload ID and Verified ID.

Key Vault
==========

.. toctree::
   :maxdepth: 1
   :glob:

   Key-Vault/*


Azure Key Vault is one of the key management solutions in Azure, and helps solve the following problems:

* Secrets Management - Use Azure Key Vault to securely store and control access to tokens, passwords, certificates, API keys, and other secrets.
* Key Management - Use Azure Key Vault as a Key Management solution. Azure Key Vault makes it easy to create and control the encryption keys used to encrypt your data.
* Certificate Management - Azure Key Vault lets you provision, manage, and deploy public and private Transport Layer Security/Secure Sockets Layer (TLS/SSL) certificates for use with Azure and your internal connected resources.

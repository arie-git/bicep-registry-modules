Container Registry
==================

.. toctree::
   :maxdepth: 1
   :glob:

   Container-Registry/*

Azure Container Registries allows you to build, store, and manage container images and artifacts in a private registry for all types of container deployments. Use Azure container registries with your existing container development and deployment pipelines. Use Azure Container Registry Tasks to build container images in Azure on-demand, or automate builds triggered by source code updates, updates to a container's base image, or timers.

Purpose
-------
This section provides relevant information about development tools usable together with the DRCP platform and its components.

DRCP-owned tools (functional owner)
-----------------------------------
Within APG, DRCP owns the following tools from functional owner perspective:

- Azure CLI
- Databricks CLI
- PowerShell Core
- OpenTofu
- Azure Storage Explorer

In case of issues, please follow the regular `self-service processes <https://apgprd.service-now.com/service_portal?id=sc_category&sys_id=56755bf9379962003486f01643990e97>`__ in ServiceNow. If your incident relates to the functional use of the tool, WPS dispatches the incident to DRCP for support. Workplace Services provides technical support for issues like package installation or updates.

In case of requesting a newer version, please use the `Ask a question <https://apgprd.service-now.com/drdc?id=drdc_sc_item&source=drdc_sc_ce_index&table=sc_category&sys_id=f99f79eedb2c98100228bc45f39619b1>`__ option in the DRDC portal, so DRCP can align your request with its LCM processes.

Release 23-2 06.1
=================
Release date: June 6, 2023


What's new for users
--------------------
- The Azure DevOps service connections to the Azure Subscription now comply to the naming standard **SC-<Application system name>-<environment code>**. Make sure you update them in your pipelines!
- Added a policy which completes the private DNS registration for Azure Files (on a Storage Account) private endpoints.

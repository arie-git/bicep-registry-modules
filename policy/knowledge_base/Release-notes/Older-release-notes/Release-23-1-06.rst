Release 23-1 06
===============

What's new for users
--------------------
- Component baselines are available for Key Vault, Cosmos DB (NoSQL), SQL and storage account.
- Feature toggles are available for Cosmos DB.
- Extended code quality check by added policy schema validation.
- Added least privilege to Application system creation pipeline.
- Added an option for more then one mail-address to Microsoft Defender.
- Added a custom Azure policy for AKS route tables.
- Added the virtual network size selector to the ServiceNow portal when requesting an environment.
- The remove Application system pipeline works.
- The reservation in Infoblox contains the Subscription id for an environment.
- The option retryCountOnTaskFailure in the create environment pipeline handles certain timeouts.

Fixed issues
------------
- Fixed an issue with overlapping policy definition names.
- Fixed an issue with the virtual network peering policy.

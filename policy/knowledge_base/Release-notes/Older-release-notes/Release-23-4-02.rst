Release 23-4 02
===============
Release date: October 24, 2023

.. warning:: Breaking change: As mentioned in the previous release notes, the privileges and roles for generating the API Keys for Application Insights ``are disabled``. `[SAA-5425] <https://jira.office01.internalcorp.net:8443/browse/SAA-5425>`__

What's new for users
--------------------

- Added the security baseline for Application Gateway (WAF). `[SAA-3941] <https://jira.office01.internalcorp.net:8443/browse/SAA-3941>`__
- Added or updated 22 App Service policies. See the Jira story for an overview of the changes and their meaning. The :doc:`baseline documentation <../../Azure-components/App-Service/Security-Baseline>` updated to reflect these changes. `[SAA-4053] <https://jira.office01.internalcorp.net:8443/browse/SAA-4053>`__
- A new ADO project S01-App-MULE-Templates was setup for integrating MULE in the DRCP Platform. `[SAA-4724] <https://jira.office01.internalcorp.net:8443/browse/SAA-4724>`__
- Added the process for enabling Zone Redundancy within Subscriptions. `[SAA-5461] <https://jira.office01.internalcorp.net:8443/browse/SAA-5461>`__

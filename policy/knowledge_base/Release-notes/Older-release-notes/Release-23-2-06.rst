Release 23-2 06
=================
Release date: May 24, 2023


What's new for users
--------------------
- The Azure built-in policies for keys, secrets, certificates and HSM keys, in a Key Vault have now a maximal validity period of 365 days.
- Added a policy which enforces a route table on NSGs.

Fixed issues
------------
- API Management is now an allowed service tag and routes through the firewall to the internet.

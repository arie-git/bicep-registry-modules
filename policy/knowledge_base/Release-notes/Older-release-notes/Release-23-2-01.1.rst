Release 23-2 01.1
=================
Release date: April 4, 2023


What's new for users
--------------------
- Updated 'DRCP Contributor' role and revoked 'User Access Administrator' role to prevent accidental policy deletion.
- Deployments from Azure DevOps can now create a change with more then one Environments related to the change.

Fixed issues
------------
- Fixed an issue with refreshing an environment when it contains an used subnet.
- Fixed an issue where deployments of environments failed when the Application system contained '-'.

Release 23-1 04
===============

What's new for users
--------------------
- The diagnostic logs are now send to the central Log Analytics workspace for Sentinel.
- All three the stacks have now a separate API account.
- Feature toggles are available for AKS, ACR and EventHub.
- Automated private DNS registration are available.
- The environment removal pipeline removes Service Connections in Azure DevOps.
- The environment creation pipeline makes network reservations in Infoblox.

Release 23-1 05
===============

What's new for users
--------------------
- Feature toggles are available for Key Vault and SQL.
- The Application system removal is available in ServiceNow portal.
- Variable groups are available to store Infoblox reservations.
- Component baselines are available for ACR.
- Automatic remediation polices for diagnostic settings are in place.
- The environment removal pipeline removes the Infoblox reservation.

Fixed issues
------------
- Fixed an issue where the policy blocked a virtual network refresh.
- Fixed an issue where search filters returned an error when no results found.

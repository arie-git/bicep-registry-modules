Release 23-2 03
====================
Release date: April 25, 2023


What's new for users
--------------------
- It's now possible to create an incident using the DRDC portal. The "Ask a question" and "Create an incident" requests contain options for the ServiceNow portal, Azure portal and, Deployments.
- It's now possible to use local authentication for Cosmos DB and Event Hub in an environment with usage development.
- Component baseline is now available for Azure App Service (scoped to Web App, Function App, Logic App v2, and API App. Mobile App is out of scope).
- It's now possible to use a Quick action that opens the Azure DevOps project related to the environment or Application system.

Fixed issues
------------
- Fixed an issue where it's possible to refresh an environment while it wasn't fully installed.

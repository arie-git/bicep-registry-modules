Application reference architectures
===================================

Purpose
-------

| Application reference architectures are quick starter solution architectures. They can help to get started with an Application system, based on Azure components provided via the DRCP platform.
| As part of the onboarding process, the BU CCC can support DevOps teams and their Product Architects to create an initial DRCP compliant solution architecture.
| If you are a member of a DevOps team with an interest in specific application reference architectures, please :doc:`contact <Need-help>` your BU CCC.
